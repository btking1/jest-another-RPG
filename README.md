## Description
   
jest another RPG is a simple RPG game that is built with the jest framework.

    - Built with: 
        * JavaScript
        * Node JS
        * Express JS
        
<p>&nbsp;</p>

## Usage

    - Install:
        * npm install
        
    - Run:
        * npm start app.js
